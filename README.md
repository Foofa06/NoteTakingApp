Hi! This is my SwiftUI note-taking app.

I built this app using SwiftUI for my indiviual assignment #2. It allows users to:

- Add new notes
- Edit existing notes
- Mark notes as completed and also incompleted
- Delete notes
- Save notes even after closing the app

Each note has:
- A title
- Content
- A completion status

The app uses JSON encoding with AppStorage to save all notes locally so they stay even after the app restarts.
